# Requirement specifications

## Purpose of the application
This application is meant to be a program to visualize different probability distributions, namely the normal distribution and simple coin toss, but other distributions are possible too depending on advanced development time.

## Users
There are only normal users, however the possibility to add an admin user remains if it is found to be appropriate.

## Basic version

### Not logged in

With the application you can visualize and manipulate different probability distributions, but to save those distributions you need to create a username.

### Logged in

After logging in the user can see their saved probability distributions and manipulate them further.
