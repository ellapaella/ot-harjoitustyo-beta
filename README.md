# Todennäköisyysohjelma (Ohjelmistotekniikka, harjoitustyö)

Todennäköisyysohjelma sovelluksen tarkoituksena on pitää kirjaa käyttäjän luomista **todennäköisyysjakaumista** ja visualisoida niitä graafisessa käyttöliittymässä.

Voit kokeilla ohjelmaa lataamalla tämän repon ja suorittamalla sen jälkeen projektin juurikansiossa komentorivillä `poetry install` ja sen jälkeen `python3 src/main.py`. `poetry run invoke start` ei tällä hetkellä toimi kunnolla.
